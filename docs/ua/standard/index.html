Відкритий міжнародний стандарт цифрового бджільництва

Єдина семантична модель та протокол для інтеграції апаратних засобів телеметрії, систем автоматизації та інструментів операційного управління розумною пасікою.

Про ідею стандарту

Сьогодні ринок цифрового бджільництва фрагментований: кожен виробник створює власні закриті пристрої та додатки. Це змушує бджолярів користуватися десятком різних програм без можливості об'єднати дані.

SmartApiary Standard (SAS) розробляється як незалежна від брендів система специфікацій. Стандарт визначає єдину інформаційну модель даних, логічних параметрів та команд управління. Будь-який пристрій чи програма, що підтримує SAS, зможе працювати в загальній екосистемі без розробки індивідуальних інтеграцій.
